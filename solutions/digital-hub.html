Democratising information

More than two-thirds of the world’s 7.6 billion inhabitants now have a phone. With the ability to contact family, transfer money and access information mobile technology has become a lifeline for the modern refugee.

App Features

Alpha Release

Powered by Kiwix

Over 250GB of educational content curated by Kiwix.

Runs on all devices

Runs on most single board computers, and is solar-powered!

Remote access

Remote access possible through 3G/4G uplink using Balena.

User Interface

Mobile First Design

ANY MOBILE DEVICE

As a progressive web-application, any mobile device with a browser and a WiFi receiver can use this application, without the need for an active internet connection.

MULTI-LANGUAGE SUPPORT

Our goal is to help provide quality digital content to underserved populations in refugee camps. Currently supported languages are English and French.
